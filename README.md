# Математический Помощник

Функции приложения:
Решение квадратных уравнений — находит корни уравнений вида ax² + bx + c = 0

Построение графиков функций — визуализирует любые математические функции (sin, cos, x² и др.)

Поиск точек пересечения функций — находит и отображает на графике точки, где f(x) = g(x)

Конвертер систем счисления — переводит числа между системами от 2 до 16
